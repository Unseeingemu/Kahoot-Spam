# Kahoot-Spam
A npm package for spamming kahoots 😏 

---
### Spam
```js
spam(pin, name, amout)
```
##### Example
```js
const Kahoot-Spam = require('Kahoot-Spam')
Kahoot-Spam = api
api.spam(123456, 'Name', 10)
```
> *This WIll Spam The Kahoot 123456 10 Times With The Name 'Name'*
